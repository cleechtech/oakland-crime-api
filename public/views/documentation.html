oakland-crime-api documentation

Everything is `GET` request

API endpoint
http://oakland-crime.herokuapp.com/api

/crimeTotals
Sends an object with all valid `crimeTypes`, `beats` and `categories`. These correspond to the police beat where the crime occured, the type of crime and the categorization of the crime. Crime type is more specific than category
/crimeCategories
Get all the categories for crimes
/beat/:beat
Crimes a given police beat from 2007 to 2013
/api?start=YYYY-MM-DD&end=YYYY-MM-DD
Gets crimes all crimes that occured within the date range. Valid from 2007 to about August 2014 **This isn't working right now cause of heroku rate limits/request times out. Fixing soon.. **
